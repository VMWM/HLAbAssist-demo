# HLAbAssist Demo

Public demo of HLAbAssist, a dashboard for standardized interpretation of HLA antibody data in kidney transplantation research.

Live site: https://vmwm.github.io/HLAbAssist-demo/

This repository contains only the built static site (HTML, JS, images). The source code is maintained in a separate private repository and deployed here as sanitized output.

The demo uses a de-identified case from the ASHI Proficiency Testing program (2023 Virtual Crossmatch Educational Challenge, Case 13) to illustrate the algorithm's reasoning, including per-rule evidence scoring, donor-specific antibody matching, and virtual crossmatch prediction.

## About the project

HLAbAssist is a predoctoral research project (Vanessa Menard, Tulane University) developing interpretable machine learning for HLA antibody interpretation. Funded by the University of Alabama at Birmingham CCTS predoctoral T32 training program and an F31 fellowship application (pending review).

For research correspondence, see the live site's About page.
